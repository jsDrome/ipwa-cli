# Setup

```shell
npm i -g jsdrome
```

# Configuration

Set the following secrets in Github and you are good to go.

  - `PROJECT_NAME`
  - `DOCKER_USER`
  - `DOCKER_PASS`
  - `DOCKER_IMAGE_NAME`
  - `HEROKU_APP_NAME`
  - `HEROKU_API_KEY`
  - `CODECOV_TOKEN`
  - `CYPRESS_TOKEN`
  - `PERCY_TOKEN`
  - `FIREBASE_TOKEN`
  - `NPM_TOKEN`
